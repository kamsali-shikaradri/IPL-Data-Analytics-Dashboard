 # IPL Data Analytics Dashboard

## Project Overview
This project analyzes IPL data using Power BI and SQL to uncover insights about teams, players, venues, and match trends.

## Tools Used
- Power BI
- SQL
- Excel

## Dashboard Features
- Total Matches Played
- Total Runs Scored
- Total Teams
- Total Wickets
- Team Wins Analysis
- Toss Impact Analysis
- Top 10 Batters by Runs
- Top 10 Wicket Takers
- Venue Analysis
- Season-wise Trends

## Key Insights
- Mumbai Indians recorded the highest number of wins.
- Wankhede Stadium hosted the most matches.
- Run scoring peaked during the early IPL seasons.
- Top batters contributed a significant share of total runs.
